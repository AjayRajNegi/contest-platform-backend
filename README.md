# Contest Platform API

This project is implementation of backend of a contest platform where creators can create contests and add MCQ, DSA questions. Any contestee can submit thier solutions.

Notion doc: https://brindle-goal-102.notion.site/Contest-Platform-Backend-Assignment-2e646b36b2e980538ef7ed491844c276#2e646b36b2e980a98299c31cca8a8596

Test Repo: https://github.com/rahul-MyGit/contest-test

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.8. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
